# Double Check Article - 中文文章检查技能

## 描述
中文文章内容检查工具，用于校对和改进中文文章质量。当用户需要检查中文文章、校对中文内容、修改错别字、优化中文表达、审核中文文档时使用此技能。

## 适用场景
1. **检查错别字和 typo** - 自动识别中文文章中的错别字和拼写错误
2. **审核表达是否清晰合理** - 检查语句是否通顺，逻辑是否清晰
3. **提供修改建议** - 给出具体的修改建议和改进方案
4. **风格一致性检查** - 确保整篇文章的风格和语气一致
5. **格式规范化** - 检查标点符号、段落格式等

## 使用方法

### 基本检查
当用户提供中文文章时，自动进行以下检查：
1. **错别字检查** - 使用常见错别字库进行匹配
2. **语法检查** - 检查句子结构是否完整
3. **表达清晰度** - 评估语句是否容易理解
4. **逻辑连贯性** - 检查段落间的逻辑关系

### 详细报告
提供结构化的检查报告：
- **严重问题**（必须修改）
- **建议改进**（推荐修改）
- **风格建议**（可选优化）
- **整体评分**（1-10分）

## 检查项目

### 1. 语言准确性
- 错别字和拼写错误
- 词语搭配是否恰当
- 成语使用是否正确
- 专业术语是否准确

### 2. 表达清晰度
- 句子是否过长或过短
- 主谓宾结构是否清晰
- 修饰语位置是否恰当
- 是否有歧义表达

### 3. 逻辑结构
- 段落主题是否明确
- 论点论据是否匹配
- 过渡是否自然
- 结论是否有力

### 4. 风格一致性
- 语气是否一致（正式/非正式）
- 人称是否统一
- 时态是否一致
- 格式是否规范

## 工作流程

### 步骤 1：接收文章
用户提供中文文章（可以直接粘贴或提供文件路径）

### 步骤 2：初步分析
- 统计文章基本信息（字数、段落数、句子数）
- 识别可能的语言问题
- 标记需要重点检查的部分

### 步骤 3：详细检查
逐段检查，重点关注：
1. **开头段落** - 是否吸引读者
2. **主体段落** - 论证是否充分
3. **结尾段落** - 总结是否有力
4. **过渡部分** - 连接是否自然

### 步骤 4：生成报告
提供：
1. **问题列表**（按严重程度排序）
2. **修改建议**（具体修改方案）
3. **优化建议**（提升文章质量的建议）
4. **最终评分**（整体质量评估）

## 示例

### 输入文章
```
人工智能正在改变世界。它可以帮助我们完成很多工作，比如数据分析、图像识别等。但是，人工智能也带来了一些问题，比如就业岗位的减少。
```

### 检查报告
```
📊 文章基本信息：
- 字数：45字
- 段落：1段
- 句子：3句

🔍 检查结果：

✅ 优点：
1. 主题明确，聚焦人工智能的影响
2. 语言简洁，表达清晰

⚠️ 建议改进：
1. 可以增加具体案例，使论述更有说服力
2. "就业岗位的减少"可以改为"就业岗位的变化"更准确
3. 可以考虑增加过渡句，使段落更连贯

📈 优化建议：
1. 增加人工智能的具体应用案例
2. 讨论人工智能带来的机遇和挑战
3. 提供平衡的观点

🎯 整体评分：7/10
```

## 配置选项

### 检查严格度
- **宽松模式**：只检查严重错误
- **标准模式**：检查所有常见问题（默认）
- **严格模式**：进行深度分析，包括风格和逻辑

### 报告格式
- **简洁报告**：只列出主要问题
- **详细报告**：包含所有检查项和建议（默认）
- **专业报告**：适合正式文档的详细分析
