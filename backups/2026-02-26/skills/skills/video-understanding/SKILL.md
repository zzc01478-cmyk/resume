# Video Understanding Skill

使用 Gemini 3 多模态模型进行视频内容分析。

## 功能
- 视频内容分析（主题、场景、情感等）
- 多模态理解（视觉+音频）
- 中文输出详细报告
- 自动调用 Gemini API

## 使用方法
当用户需要分析视频时，自动调用此技能。

### 触发条件
- 用户提到"分析视频"、"视频理解"、"看懂视频"
- 用户提供视频文件或链接
- 用户询问视频内容

## 配置
需要设置 Gemini API key：
1. 将 API key 存储在 `config/gemini-config.json` 中
2. 确保 `video_understanding_enabled` 为 true

## 工作流程
1. 接收视频输入（文件路径或URL）
2. 读取 Gemini 配置
3. 调用 Gemini API 进行视频分析
4. 返回结构化分析报告
